# Cloud Hosting Service Landing Page

## What does this do?

It provides a responsive cloud hosting landing page with an infrastructure preview, feature cards, and reliability metrics.

## How is it used?

```html
<section class="hero">
  <h1>
    Deploy apps globally with autoscaling, observability, and predictable costs.
  </h1>
  <a class="button" href="#infra">Deploy project</a>
</section>
```

## Why is it useful?

It demonstrates a complete hosting product page with CSS-only infrastructure motion and clear technical value sections.
