# HR Payroll SaaS Landing Page

## What does this do?

It provides a responsive HR and payroll SaaS landing page with a payroll summary, people operations cards, and workflow states.

## How is it used?

```html
<section class="hero">
  <h1>
    Run payroll, time-off, benefits, and employee records from one system.
  </h1>
  <a class="button" href="#payroll">Run payroll</a>
</section>
```

## Why is it useful?

It demonstrates a complete HR product page with subtle CSS motion for payroll readiness and operations workflow.
