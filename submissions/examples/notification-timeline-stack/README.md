# Notification Timeline Stack

## What does this do?
A stacked notification timeline with animated entry cards and pulsing status dots.

## How is it used?
```html
<div class="notification-stack">
  <article><span></span><strong>Build completed</strong></article>
</div>
```

## Why is it useful?
It gives EaseMotion CSS a timeline-style notification pattern for deployments, activity feeds, and review queues.
