# CRM SaaS Landing Page

## What does this do?

It provides a responsive CRM SaaS landing page with a pipeline preview, sales feature cards, and forecast metrics.

## How is it used?

```html
<section class="hero">
  <h1>Track every lead, handoff, and deal stage without losing context.</h1>
  <a class="button" href="#pipeline">View pipeline</a>
</section>
```

## Why is it useful?

It demonstrates a complete sales product page with lightweight CSS animation for deal movement and status emphasis.
