# Pricing Toggle Switcher

## What does this do?
A billing-cycle toggle with an animated active pill and a responsive pricing panel.

## How is it used?
```html
<div class="billing-toggle">
  <button class="active">Monthly</button>
  <button>Yearly</button>
</div>
```

## Why is it useful?
It gives EaseMotion CSS a common SaaS pricing control with clear motion and no JavaScript dependency.
